# 자바 기초 개념 정리

---

## 1. 클래스, 객체, 메인

- **클래스(Class)** : 객체를 만들기 위한 **설계도**  
- **객체(Object)** : 클래스로부터 생성된 **실제 실체**, 필드와 메서드를 가짐  
- **메인(main)** : 프로그램 실행 시작점.  
  - 객체 생성  
  - 필드에 값 넣기  
  - 메서드 호출 → 프로그램 동작

---

## 2. 메서드(Method)와 필드(Field)

- **메서드(Method)** : 객체가 수행할 **기능, 동작**을 정의  
- **필드(Field, 멤버 변수)** : 객체가 가지는 **데이터, 상태**를 저장

---

## 3. return 키워드

1. **값을 반환하는 메서드에서 사용**  
   - 반환값이 있는 메서드는 반드시 `return 값;` 필요  

2. **메서드를 조기 종료할 때 사용**  
   - 반환값이 없는 `void` 메서드에서도 `return;` 가능  

---

## 4. 점(dot) 연산자

- **객체의 변수나 메서드에 접근할 때 사용**

```java
객체이름.변수   // 객체 안의 필드에 접근
객체이름.메서드() // 객체의 기능(메서드) 실행
````

---

## 5. 배열(Array) 문법

* 기본 선언

```java
타입[] 변수명;
```

* 예시 1: 정수 배열

```java
int[] a = {1, 2, 3, 4};
```

* 예시 2: 객체 배열

```java
Course[] courses = new Course[5];
```

* **설명**

  * 크기 5짜리 배열 생성
  * 배열 안에는 아직 객체가 들어있지 않고, `null` 값이 들어 있음

---

## 6. 메서드의 파라미터(Parameter)

* **파라미터** : 메서드를 호출할 때 전달되는 값(인자, argument)을 받는 변수
* **타입 지정 이유** : 컴파일러에게 어떤 종류의 값이 들어오는지 명시하기 위함

```java
public int sum(int a, int b) {
    return a + b;
}
```

* `int a, int b` → 파라미터, 호출 시 전달된 값이 `a`, `b`에 들어감

---

## 7. 람다식(Lambda Expression)

* **형식**

```java
(매개변수 목록) -> { 실행문 }
```

* 예시

```java
Runnable r = () -> System.out.println("스레드 실행!");
r.run();
```

* 람다는 **단일 추상 메서드(SAM)를 가진 인터페이스**를 간단하게 구현할 때 사용

---

## 8. 향상된 for문(Enhanced for loop)

* 배열이나 컬렉션을 **간결하게 순회(iterate)** 할 때 사용

```java
for (타입 변수 : 배열_또는_컬렉션) {
    // 변수 사용
}
```

* 예시: 배열 합계 계산

```java
int[] numbers = {10, 20, 30, 40};
int sum = 0;

for (int num : numbers) {
    sum += num; // num 값을 sum에 더함
}

System.out.println("총합: " + sum);
```

* **주의점**

  * 배열 요소 자체를 바꾸는 건 불가능 (기본형 배열)
  * 객체 배열일 경우 **필드 값 변경은 가능**
